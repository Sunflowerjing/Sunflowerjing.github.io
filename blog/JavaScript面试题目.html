  1. 请问变量a会被GC回收么，为什么呢? 不会被回收
function test(){
    var a = "yideng"; 
    return function(){
        eval("");  // eval 形成了闭包
    }
} 
test()();

答案: 不会被回收, 因为eval

(1) 闭包: 保留住函数执行的作用域 (2) 函数执行: 会创造执行栈(call stack) (3) anonymous: 匿名的 (4) GC 不会实时进行回收的 (5) 若要回收 则 const = test()(); x = null
