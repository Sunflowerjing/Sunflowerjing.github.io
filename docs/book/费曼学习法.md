# 费曼学习法

1. 困扰: 表达无逻辑、概念不清楚、新知识不清楚
2. 解决办法
    * 把学到的概念和新知识写在纸上
    * 用最简单的语言讲给别人听, 在二次复述的时候着重练习自己卡住的地方
    * 把自己巩固和强化过的概念再次转述给别人
    * 提炼精华, 精简表达概念
3. 步骤
    * 第一步: 选择目标: 选择想要学习的概念
    * 第二步: 教学: 设想, 你面对这个领域的菜鸟, 试图解释清楚这个概念, 并且让对方完全听懂
    * 第三步: 纠正学习: 查漏补缺, 当无法解释的时候, 重新回头找答案, 回到书上去、找同学老师。找已经懂的人, 把这个概念重新研究一遍。
    * 第四步: 简化讲解

4. 核心: 用自己的语言来讲述, 你要学习的概念
