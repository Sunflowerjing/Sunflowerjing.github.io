# 联动菜单

## 两个select联动
* 选择**第一个下拉框**, 第二个里面的**值跟着相应的变化**

1. 初始化数据
```javascript
<!-- 一级 select 数据  -->
const mainData = [
    '水果', '玩具'
];
<!-- 二级 select 数据 -->
const secondData = {
    '水果': ['西瓜', '香蕉', '葡萄'],
    '玩具': ['水枪', '泡泡机']
}
```
2. state 中的数据
```javascript
state = {
    main: secondData[mainData[0]],  // 一级
    second: secondData[mainData[0]][0]  // 二级
}
```
3. 修改数据
```javascript
<!-- 修改一级数据  -->
handleMain = ({target:{value}}) => {
    this.setState({
        main: secondData[value],
        second: secondData[value][0]
    })
}

<!-- 修改二级数据  -->
handleSecond = ({target:{value}}) => {
    this.setState({
        second: value
    })
}
```

4. render 内容
```javascript
<div>
    <p>两个select联动菜单</p>
    <select
        style={{ width: 120 }}
        defaultValue={mainData[0]}
        onChange={(e) => this.handleMain(e)}
    >
        {
            mainData.map((mainItem) => {
                return <option key={mainItem}>{mainItem}</option>
            })
        }
    </select>
    <select
        style={{ width: 120 }}
        value={this.state.second}
        onChange={(e) => this.handleSecond(e)}
    >
        {
            this.state.main.map((mainItem) => {
                return <option key={mainItem}>{mainItem}</option>
            })
        }
    </select>
</div>
```
